# Security Policy

## Supported Versions

Currently, only the latest release of the `gruff` ecosystem is actively supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this repository, please do **not** open a public issue.

Instead, please send an email to the repository owner at **irfankabir02@gmail.com**. We will strive to acknowledge your report within 48 hours and provide a timeline for a fix. We ask that you do not publicly disclose the issue until a patch has been released.

**Network Isolation Note**: As per TUV-001 (The Unbreakable Vow), all components in this repository operate under strict network isolation (localhost only). If you find any capability that circumvents this or exfiltrates telemetry data, please report it immediately as a critical vulnerability.
