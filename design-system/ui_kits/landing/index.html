What GRID does

Built for engineers who want to know what the code actually does.

Not a chatbot wrapper. A local-first framework with typed tokens, explicit patterns, and a console that admits when it doesn't know.

Local-first, by default

Your code never leaves your machine. Models run through Ollama; the index lives on disk. No telemetry.

9 Cognition Patterns

Admission, Resonance, Constraint, Pivot, Gate, Compress, Signal, MIST, Fabric — composable reasoning primitives.

Deterministic by design

Pinned versions, cached embeddings, reproducible runs. Same question, same answer — or a documented reason.

+37% precision on RAG

Geometric resonance retrieval beats naive embedding search by 33–40% on the internal eval set.

Admits what it doesn't know

MIST flags answers with high confidence we don't know. No hallucinated citations, no confident filler.

Works on real codebases

Tested against 200K-line monorepos, Rust workspaces, Python monoliths. 1,490 files indexed in four seconds.

The nine

Cognition patterns, not prompts.

Typed reasoning primitives. Composable. Inspectable. Each one earns its place.

01

Admission

Gate new context before ingestion. No signal in, no noise out.

02

Resonance

Match by structural similarity, not token overlap.

03

Constraint

Hard-bound outputs to the schema that the caller declared.

04

Pivot

Swap reasoning axis when confidence drops below threshold.

05

Gate

Conditional forwarding with typed predicates.

06

Compress

Lossy, principled reduction to a structured core.

07

Signal

Separate meaningful change from noise before commit.

08

MIST

High confidence that we don't know. Epistemic humility, typed.

09

Fabric

The woven index of all the above. The memory that persists.

One command

Point it at a repo. Ask a question.

GRID indexes on first run, caches on disk, and answers in seconds. Prefer the UI? There's one of those too.

$ grid analyze ./src --local → 1,490 files · 4.2s · 37% precision lift   $ grid ask "where does admission happen?" → src/patterns/admission.py:42 → called from orchestrator.py:118, fabric.py:77 → confidence: high · MIST: no   $ grid status ollama · online · llama3.2:3b index · 128MB · fresh
The story

GRID is built by someone who cares about doing things right — principled, not perfect.

Six months, one engineer, 200,000 lines. The goal was never to build a product. It was to build a thing that would still make sense in a year. That meant typed tokens, pinned versions, and a console that admits when it doesn't know. It meant writing the 4,490th test before shipping the feature. It meant throwing away the clever version when the plain version was clearer.

The amber wheel is the metaphor. Eight spokes, one hub. Many patterns, one fabric. Separate signal from noise, compress it into a structured core, and keep moving.

— caraxesthebloodwyrm02, solo maintainer
